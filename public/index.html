// Challenge Brief

You've reached the Ultimate Revenge operator terminal — a classified system locked behind three interlocking security mechanisms. Each vulnerability you discover unlocks the next layer. The final flag is assembled from three fragments scattered across the system. No single exploit is enough; you must chain all three in order.

01
Authenticate

Locate valid credentials and obtain a session token. The JWT payload itself holds no flag — but without it, the rest of the system is dark.

02
Traverse

The file reader is guarded, but its filters are flawed. Flag fragments are stored in files named .p1 and .p2 outside the app root. The RSA public key lives under /tmp/.cache/ with a .key extension — its exact name is dynamic.

03
Escalate

The signing algorithm can be turned against itself. A public key used as an HMAC secret produces tokens the server will accept. Hit the admin endpoint with an insufficient token first — it will tell you exactly which claims you are missing.
